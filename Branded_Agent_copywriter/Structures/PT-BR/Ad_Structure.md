# Estrutura: Anúncio

> Template para conteúdo promocional direto.
> Ideal para: lançamentos, ofertas, lead magnets, produtos.

---

## Fórmula Base

```
OBJEÇÃO / OBSTÁCULO (reconhecimento da dor)
    ↓
ALTERNATIVA INTELIGENTE (a solução)
    ↓
PRÉ-CTA (quando couber)
    ↓
CTA DIRETO (ação específica)
```

---

## Timeline Sugerida (30-45s)

| Tempo | Elemento | Descrição |
|-------|----------|-----------|
| 0:00-0:05 | **Hook** | Chama a objeção principal |
| 0:06-0:12 | **Agitação** | Por que isso dói / consequências |
| 0:13-0:25 | **Solução** | A alternativa / o que você oferece |
| 0:26-0:35 | **Prova** | Resultado, número ou depoimento |
| 0:36-0:45 | **CTA** | Ação específica + urgência |

---

## Tipos de Anúncio

### Anúncio de Dor (Pain-focused)
```
"Se você tá [dor], para tudo."
    ↓
"O problema não é [causa óbvia]. É [causa real]."
    ↓
"Por isso eu criei [solução]."
    ↓
"[Número] pessoas já [resultado]."
    ↓
"[CTA com link/ação]."
```

### Anúncio de Resultado (Outcome-focused)
```
"[Resultado específico] em [tempo]."
    ↓
"Sem [sacrifício que o público odeia]."
    ↓
"Usando [método/ferramenta]."
    ↓
"Funciona pra [perfil específico]."
    ↓
"[CTA]."
```

### Anúncio de Objeção (Objection-focused)
```
"Eu sei que você acha que [objeção]."
    ↓
"Mas [contra-argumento]."
    ↓
"Por isso [solução funciona assim]."
    ↓
"[CTA]."
```

---

## Elementos Obrigatórios

### Hook (0:00-0:05)
- [ ] Chama a atenção do público certo
- [ ] Filtra quem não é o ICP
- [ ] Reconhece uma dor real

### Corpo (0:06-0:25)
- [ ] Agita a dor (consequências)
- [ ] Apresenta alternativa clara
- [ ] Diferencia de soluções comuns

### Prova (0:26-0:35)
- [ ] Número específico
- [ ] Depoimento curto
- [ ] Antes/depois visual

### CTA (0:36-0:45)
- [ ] Ação específica ("clique", "comente", "acesse")
- [ ] Urgência (quando aplicável)
- [ ] Baixa fricção (fácil de executar)

---

## CTAs para Anúncios

### Link Externo
- "Link na bio."
- "Clica no link e garante o seu."
- "Acessa [nome do link] e vê como funciona."

### Lead Magnet
- "Comenta [PALAVRA] que eu te mando."
- "Salva e me chama no direct."
- "Deixa seu email nos comentários."

### Escassez
- "Últimas [X] vagas."
- "Só até [data]."
- "Pra quem ver esse vídeo primeiro."

---

## Variações

### Anúncio Ultra-Curto (15s)
```
Hook + dor [0:00-0:05]
Solução em 1 frase [0:06-0:10]
CTA direto [0:11-0:15]
```

### Anúncio Longo (60s)
```
Hook [0:00-0:05]
História rápida [0:06-0:15]
Problema [0:16-0:25]
Solução [0:26-0:40]
Prova [0:41-0:50]
CTA + urgência [0:51-1:00]
```

---

## Checklist Final

- [ ] Hook qualifica o público certo?
- [ ] A dor é real e específica?
- [ ] A solução é clara e diferenciada?
- [ ] Tem prova (número, depoimento, resultado)?
- [ ] CTA tem baixa fricção?
- [ ] Urgência é real (não falsa)?
