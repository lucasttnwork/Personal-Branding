# Estrutura: Storytelling

> Template para roteiros narrativos com história pessoal.
> Ideal para: confissões, momentos de virada, casos reais, lições aprendidas.

---

## Fórmula Base (Midtrack)

```
HOOK (vulnerabilidade / cena específica)
    ↓
BACKSTORY (contexto curto)
    ↓
PONTO DE QUEBRA (momento zoom-in)
    ↓
VIRADA (insight / mudança)
    ↓
TAKEAWAY (lição aplicável)
    ↓
SOFT CTA (pergunta + convite)
```

---

## Timeline Sugerida (90s)

| Tempo | Elemento | Descrição |
|-------|----------|-----------|
| 0:00-0:05 | **QPV1 + Hook** | Cena específica ou confissão forte |
| 0:06-0:20 | **Backstory** | Contexto: onde, quando, o que estava acontecendo |
| 0:21-0:35 | **QPV2 + Tensão** | O problema no auge — antes da virada |
| 0:36-0:55 | **Ponto de Quebra** | Momento exato: frase, cena, evento específico |
| 0:56-1:10 | **QPV3 + Virada** | O que mudou — o insight |
| 1:11-1:20 | **Takeaway** | Lição aplicável ao espectador |
| 1:21-1:30 | **Soft CTA** | Pergunta reflexiva + convite |

---

## Elementos Obrigatórios

### Hook (0:00-0:05)
- [ ] Vulnerabilidade ou cena específica
- [ ] Detalhes sensoriais (lugar, hora, frase)
- [ ] QPV que cria imersão

### Backstory (0:06-0:20)
- [ ] Contexto suficiente para entender a tensão
- [ ] Não mais que 15 segundos
- [ ] Tom conversacional, como desabafo

### Ponto de Quebra (0:21-0:55)
- [ ] Momento específico com zoom-in
- [ ] Uma frase ou evento que mudou tudo
- [ ] QPV de impacto emocional

### Virada + Takeaway (0:56-1:20)
- [ ] O que você entendeu/aprendeu
- [ ] Conexão com a dor do espectador
- [ ] Metodologia ou framework (se aplicável)

### Soft CTA (1:21-1:30)
- [ ] Pergunta que força auto-reflexão
- [ ] Convite, não ordem
- [ ] Opção de palavra-chave para engajamento

## Variações

### Story Curto (60s)
```
Hook + Cena [0:00-0:05]
Contexto mínimo [0:06-0:12]
Tensão → Quebra [0:13-0:30]
Virada rápida [0:31-0:45]
Pergunta + CTA [0:46-0:60]
```

### Story Longo (120s)
```
Hook cinematográfico [0:00-0:08]
Backstory detalhado [0:09-0:30]
Tensão expandida [0:31-0:50]
Momento de quebra [0:51-1:10]
Virada + método [1:11-1:40]
Takeaway + pergunta [1:41-1:55]
CTA com escassez [1:56-2:00]
```

---

## Dicas de Execução

### Tom de Voz
- Mais lento que infotenimento
- Pausas estratégicas nos pontos emocionais
- Olhar direto para câmera nos momentos de vulnerabilidade

### QPVs para Storytelling
- Flashbacks (mudança de iluminação/cor)
- Objetos que representam a história (foto, objeto antigo)
- Match cuts entre passado e presente
- Freeze frames em momentos-chave

---

## Checklist Final

- [ ] Tem cena específica (lugar, hora, frase)?
- [ ] O backstory é curto e suficiente?
- [ ] O ponto de quebra é um momento, não explicação?
- [ ] A virada é genuína (não forçada)?
- [ ] O takeaway é aplicável ao espectador?
- [ ] O CTA é suave e reflexivo?
