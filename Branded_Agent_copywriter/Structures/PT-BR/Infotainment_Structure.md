# Estrutura: Infotenimento

> Template para roteiros de conteúdo educativo + entretenimento.
> Ideal para: dicas rápidas, myth-busting, testes, revelações.

---

## Fórmula Base

```
ERRO COMUM / CRENÇA LIMITANTE
    ↓
SOLUÇÃO SIMPLES / REVELAÇÃO
    ↓
PRÉ-CTA (opcional, no meio)
    ↓
PERGUNTA ANTES DO CTA
    ↓
CTA PADRÃO OU LOOPING
```

---

## Timeline Sugerida (60s)

| Tempo | Elemento | Descrição |
|-------|----------|-----------|
| 0:00-0:03 | **QPV1 + Hook** | Gancho provocativo + quebra visual forte |
| 0:04-0:12 | **Problema** | Expande o erro/crença — por que isso dói |
| 0:13-0:23 | **QPV2 + Analogia** | Metáfora visual que cristaliza o conceito |
| 0:24-0:40 | **Solução** | O que fazer diferente (passos ou teste) |
| 0:41-0:50 | **QPV3 + Diagnóstico** | Nomeia o problema (ex: Dissonância Digital) |
| 0:51-0:60 | **QPV4 + CTA** | Pergunta + ação + loop opcional |

---

## Elementos Obrigatórios

### Hook (0:00-0:03)
- [ ] Promessa clara ou provocação
- [ ] QPV de impacto visual
- [ ] Máximo 2 frases

### Corpo (0:04-0:40)
- [ ] Contraste erro vs. acerto
- [ ] Analogia ou metáfora visual
- [ ] Passo a passo ou teste rápido
- [ ] Mínimo 2 QPVs adicionais

### Fechamento (0:41-0:60)
- [ ] Diagnóstico nomeado (terminologia própria)
- [ ] Pergunta reflexiva
- [ ] CTA específico (palavra-chave)
- [ ] Opção de loop para rewatch

## Checklist Final

- [ ] Hook promete algo específico?
- [ ] Existe contraste claro (erro vs. acerto)?
- [ ] Tem frase printável?
- [ ] QPVs distribuídos a cada 15-20s?
- [ ] CTA parece conversa, não propaganda?
- [ ] É gravável hoje com recursos disponíveis?
