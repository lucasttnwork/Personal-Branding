# Structure: Infotainment

> Template for educational + entertainment content scripts.
> Ideal for: quick tips, myth-busting, tests, revelations.

---

## Base Formula

```
COMMON MISTAKE / LIMITING BELIEF
    ↓
SIMPLE SOLUTION / REVELATION
    ↓
PRE-CTA (optional, mid-video)
    ↓
QUESTION BEFORE CTA
    ↓
STANDARD OR LOOPING CTA
```

---

## Suggested Timeline (60s)

| Time | Element | Description |
|------|---------|-------------|
| 0:00-0:03 | **VPB1 + Hook** | Provocative hook + strong visual break |
| 0:04-0:12 | **Problem** | Expand the mistake/belief — why it hurts |
| 0:13-0:23 | **VPB2 + Analogy** | Visual metaphor that crystallizes the concept |
| 0:24-0:40 | **Solution** | What to do differently (steps or test) |
| 0:41-0:50 | **VPB3 + Diagnosis** | Name the problem (e.g., Strategic Digital Dissonance) |
| 0:51-0:60 | **VPB4 + CTA** | Question + action + optional loop |

---

## Required Elements

### Hook (0:00-0:03)
- [ ] Clear promise or provocation
- [ ] High-impact VPB
- [ ] Maximum 2 sentences

### Body (0:04-0:40)
- [ ] Mistake vs. right way contrast
- [ ] Visual analogy or metaphor
- [ ] Step-by-step or quick test
- [ ] Minimum 2 additional VPBs

### Closing (0:41-0:60)
- [ ] Named diagnosis (your own terminology)
- [ ] Reflective question
- [ ] Specific CTA (keyword)
- [ ] Loop option for rewatch

---

## Variations

### Short Infotainment (30s)
```
Hook + VPB1 [0:00-0:03]
Quick problem [0:04-0:10]
Solution in 1 sentence [0:11-0:20]
CTA + Loop [0:21-0:30]
```

### Long Infotainment (90s)
```
Hook + VPB1 [0:00-0:05]
Expanded problem [0:06-0:20]
PRE-CTA [0:21-0:25]
Detailed solution [0:26-0:55]
3 quick examples [0:56-1:15]
Diagnosis + CTA [1:16-1:30]
```

---

## Final Checklist

- [ ] Hook promises something specific?
- [ ] Clear contrast exists (mistake vs. right way)?
- [ ] Has a screenshot-worthy line?
- [ ] VPBs distributed every 15-20s?
- [ ] CTA feels like conversation, not pitch?
- [ ] Shootable today with available resources?
