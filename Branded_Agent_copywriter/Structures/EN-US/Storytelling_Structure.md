# Structure: Storytelling

> Template for narrative scripts with personal story.
> Ideal for: confessions, turning points, real cases, lessons learned.

---

## Base Formula (Midtrack)

```
HOOK (vulnerability / specific scene)
    ↓
BACKSTORY (short context)
    ↓
BREAKING POINT (zoom-in moment)
    ↓
TURNING POINT (insight / change)
    ↓
TAKEAWAY (applicable lesson)
    ↓
SOFT CTA (question + invitation)
```

---

## Suggested Timeline (90s)

| Time | Element | Description |
|------|---------|-------------|
| 0:00-0:05 | **VPB1 + Hook** | Specific scene or strong confession |
| 0:06-0:20 | **Backstory** | Context: where, when, what was happening |
| 0:21-0:35 | **VPB2 + Tension** | Problem at its peak — before the turn |
| 0:36-0:55 | **Breaking Point** | Exact moment: phrase, scene, specific event |
| 0:56-1:10 | **VPB3 + Turning Point** | What changed — the insight |
| 1:11-1:20 | **Takeaway** | Lesson applicable to the viewer |
| 1:21-1:30 | **Soft CTA** | Reflective question + invitation |

---

## Required Elements

### Hook (0:00-0:05)
- [ ] Vulnerability or specific scene
- [ ] Sensory details (place, time, phrase)
- [ ] VPB that creates immersion

### Backstory (0:06-0:20)
- [ ] Enough context to understand the tension
- [ ] No more than 15 seconds
- [ ] Conversational tone, like venting

### Breaking Point (0:21-0:55)
- [ ] Specific moment with zoom-in
- [ ] One phrase or event that changed everything
- [ ] Emotionally impactful VPB

### Turning Point + Takeaway (0:56-1:20)
- [ ] What you understood/learned
- [ ] Connection to viewer's pain
- [ ] Methodology or framework (if applicable)

### Soft CTA (1:21-1:30)
- [ ] Question that forces self-reflection
- [ ] Invitation, not command
- [ ] Keyword option for engagement

---

## Variations

### Short Story (60s)
```
Hook + Scene [0:00-0:05]
Minimal context [0:06-0:12]
Tension → Break [0:13-0:30]
Quick turning point [0:31-0:45]
Question + CTA [0:46-0:60]
```

### Long Story (120s)
```
Cinematic hook [0:00-0:08]
Detailed backstory [0:09-0:30]
Expanded tension [0:31-0:50]
Breaking moment [0:51-1:10]
Turning point + method [1:11-1:40]
Takeaway + question [1:41-1:55]
CTA with scarcity [1:56-2:00]
```

---

## Execution Tips

### Tone of Voice
- Slower than infotainment
- Strategic pauses at emotional points
- Direct eye contact at vulnerability moments

### VPBs for Storytelling
- Flashbacks (lighting/color change)
- Objects that represent the story (photo, old object)
- Match cuts between past and present
- Freeze frames at key moments

---

## Final Checklist

- [ ] Has specific scene (place, time, phrase)?
- [ ] Backstory is short and sufficient?
- [ ] Breaking point is a moment, not explanation?
- [ ] Turning point is genuine (not forced)?
- [ ] Takeaway is applicable to viewer?
- [ ] CTA is soft and reflective?
