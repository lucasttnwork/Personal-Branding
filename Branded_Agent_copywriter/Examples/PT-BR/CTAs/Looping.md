# CTAs — Looping

> **Como usar:** CTAs de looping fecham o vídeo conectando ao hook inicial.
> Criam sensação de "recomeço" que incentiva rewatch automático.

## Como Funciona o Loop

### Estrutura Visual
```
HOOK: "Você tá fazendo X do jeito mais caro..."
  ↓
CONTEÚDO
  ↓
LOOP CTA: "E é por isso que eu comecei dizendo: você tá fazendo X do jeito mais caro..."
  ↓
[CORTE SECO para o início]
```

### Efeito Psicológico
- O espectador "cai" no início sem perceber
- Aumenta tempo de visualização
- Algoritmo interpreta como conteúdo de alta retenção

---

## Tipos de Loop

### Loop Literal
Repete a frase exata do hook:
- "E é por isso que eu comecei dizendo: [GANCHO]…"

### Loop Conceitual
Conecta a conclusão ao tema inicial:
- "E foi aí que eu percebi: [GANCHO]…"

### Loop Pergunta
Deixa uma questão que remete ao início:
- "E a pergunta que eu deixo é: [GANCHO]?"

### Loop Desafio
Provoca o espectador a revisitar:
- "Agora volta no começo e vê de novo com essa lente."

---

## Execução Técnica

1. **Grave o final olhando para câmera** (como no hook)
2. **Use o mesmo tom de voz do início**
3. **Corte seco, sem fade** — direto para o frame inicial
4. **Sincronize o ritmo** — a última sílaba deve "emendar" na primeira

---

## Dicas de Uso

- Loop funciona melhor em vídeos curtos (30-60s)
- O hook precisa ser forte o suficiente para suportar a repetição
- Não use loop se o conteúdo não fechar o ciclo naturalmente
- Combine com Pré-CTA no meio para engajamento duplo
