# Estrutura: Série

> Template para conteúdo episódico com continuidade.
> Ideal para: tutoriais em partes, building in public, jornadas, frameworks.

---

## Fórmula Base

```
CADA EPISÓDIO:
    ↓
1 INSIGHT COMPLETO (valor independente)
    ↓
1 PERGUNTA ABERTA (gancho para o próximo)
```

---

## Estrutura de Episódio (60s cada)

| Tempo | Elemento | Descrição |
|-------|----------|-----------|
| 0:00-0:05 | **Contexto** | "Parte X de Y" + resumo de 1 frase |
| 0:06-0:12 | **Hook do Episódio** | O que esse episódio vai resolver |
| 0:13-0:40 | **Conteúdo** | O insight completo |
| 0:41-0:50 | **Takeaway** | O que levar desse episódio |
| 0:51-0:60 | **Gancho** | Pergunta que só o próximo responde |

---

## Planejamento da Série (5 Episódios)

### Episódio 1: O Problema
```
Hook: Apresenta a dor principal
Conteúdo: Por que isso acontece / diagnóstico
Gancho: "Mas como resolver?" → Ep. 2
```

### Episódio 2: O Framework
```
Hook: "Ontem falei do problema, hoje: a estrutura"
Conteúdo: Visão geral do método/framework
Gancho: "O primeiro passo é..." → Ep. 3
```

### Episódio 3: Passo 1
```
Hook: Primeiro passo do framework
Conteúdo: Como executar + exemplo
Gancho: "Feito isso, vem o que complica..." → Ep. 4
```

### Episódio 4: Passo 2-3
```
Hook: Os passos que mais geram dúvida
Conteúdo: Execução detalhada
Gancho: "E agora, a parte que ninguém fala..." → Ep. 5
```

### Episódio 5: Resultado + CTA
```
Hook: A última peça do quebra-cabeça
Conteúdo: Resultado esperado + prova
CTA Final: Convite para ação maior
```

---

## Elementos de Continuidade

### Abertura Padrão
- Número do episódio visível
- Resumo de 1 frase do anterior
- Hook específico do episódio

### Fechamento Padrão
- Takeaway claro
- Pergunta que conecta ao próximo
- CTA de engajamento (salvar, comentar)

### Identidade Visual
- Mesma paleta de cores
- Mesmo estilo de texto na tela
- Mesma estrutura de QPVs

---

## Variações

### Série Curta (3 episódios)
```
Ep. 1: Problema + Diagnóstico
Ep. 2: Solução + Método
Ep. 3: Execução + Resultado
```

### Série Longa (10 episódios)
```
Ep. 1-2: Contexto e problema
Ep. 3-4: Framework geral
Ep. 5-8: Cada passo em detalhe
Ep. 9: Erros comuns
Ep. 10: Resultado + próximos passos
```

### Building in Public
```
Cada episódio = 1 dia/semana de progresso
Formato: O que fiz → O que aprendi → Próximo passo
Gancho: "Amanhã eu vou..."
```

---

## Checklist por Episódio

- [ ] Funciona sozinho (mesmo sem ver outros)?
- [ ] Tem conexão clara com o anterior?
- [ ] Entrega 1 insight completo?
- [ ] Termina com pergunta aberta?
- [ ] Identidade visual consistente?
- [ ] Tempo similar aos outros episódios?

---

## Dicas de Execução

### Gravação
- Grave vários episódios no mesmo dia
- Use mesma roupa/cenário para continuidade
- Grave ganchos separadamente (edição mais fácil)

### Publicação
- Intervalo ideal: 24-48h entre episódios
- Fixe a playlist/série nos destaques
- Cross-reference nos comentários
